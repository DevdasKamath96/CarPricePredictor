1. Create a new anaconda env (carPrediction) with python 3.7 -- conda create -n carPrediction python=3.7
2. activate the new env.
3 Move to the directory containing the csv files where you want to create your model and type jupyter notebook.
4. Create/train the model
5. pickle the model
6. create a requirement.txt file.
>pip freeze > requirement.txt
7. install all the packages required.
8. run $ python app.py
